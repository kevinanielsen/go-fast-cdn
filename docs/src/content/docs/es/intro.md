# Introducción a go-fast-cdn

go-fast-cdn es una red de distribución de contenidos (CDN) construida con Go diseñada para servir archivos estáticos de forma rápida y eficiente.

## Características Principales
- **Alto rendimiento**: Sirve archivos con baja latencia
- **Fácil configuración**: Configuración mínima requerida
- **Almacenamiento en caché**: Caché inteligente para contenido estático
- **Multiplataforma**: Funciona en Windows, Linux y macOS

## Empezando
Para comenzar con go-fast-cdn, consulte la guía [Primeros pasos](./getting-started/installation).
