# Instalación

Esta guía le ayudará a instalar go-fast-cdn en su sistema.

## Prerrequisitos
- Go 1.19 o superior
- Git instalado

## Pasos de Instalación
## 
### 1. Clonar el Repositorio

git clone https://github.com/kevinanielsen/go-fast-cdn.git
cd go-fast-cdn
### 2. Construir la Aplicación

go build ./cmd/server
### 3. Configurar
Copie el archivo de configuración de ejemplo:


cp config.example.yaml config.yaml
### 4. Ejecutar

./server
